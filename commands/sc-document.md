# /sc:document — Generate Documentation

Generate or update project documentation from code and specs.

## What It Does

1. Generate/update API documentation from code
2. Generate/update DEPLOY.md via @deploy-guide agent
3. Update README with current state
4. Generate architecture diagram if missing
5. Verify all docs reference correct file paths

## Output
- Updated docs/ directory
- DEPLOY.md with copy-pasteable commands
